📄 Software Requirement Specification (SRS)
📱 Project Title: Plant Watering Reminder App (Flutter + Firebase)
1. 📌 Introduction
1.1 Purpose

The purpose of this application is to help plant lovers efficiently manage and track watering schedules of their plants. The system provides reminders, stores plant data, and ensures users never miss watering their plants.

1.2 Scope

This application allows users to:

Add and manage plants
Track watering schedules
Receive reminders
Store data locally using JSON
Sync data with Firebase Firestore

The system is developed using Flutter for cross-platform mobile support.

1.3 Definitions
Watering Frequency: Number of days between watering
Firestore: Cloud database by Firebase
JSON Storage: Local file storage format
2. 🧭 Overall Description
2.1 Product Perspective

This is a standalone mobile application with optional cloud backup using Firebase.

2.2 Product Functions

The system performs the following:

Add plant details
Calculate next watering date
Show reminders
Store and sync plant data
Display watering schedule
2.3 User Characteristics
Basic smartphone users
No technical knowledge required
Suitable for plant lovers, beginners, and gardeners
2.4 Constraints
Must be developed using Flutter only
Requires internet for Firebase sync
Limited storage for local JSON

3.1 Add Plant

Watering frequency
3.2 View Plants
Display list of all plants
Show next watering date

3.4 Delete Plant

3.5 Watering Reminder
Notify user when watering is due

Store plant data in JSON file
3.7 Firebase Sync
Backup plant data to Firestore

Show upcoming watering tasks in list format
